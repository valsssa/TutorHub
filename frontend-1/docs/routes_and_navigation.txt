
# Routes & Navigation Map

EduConnect uses a custom internal routing system managed by `NavigationContext` and `AppRouter`. The application state `currentView` determines which page component is rendered.

## ViewState Definition
Defined in `domain/types.ts`.

| ViewState | Page Component | Description |
| :--- | :--- | :--- |
| `home` | `MarketplacePage` | The main landing page and tutor search grid. |
| `login` | `LoginPage` | Authentication forms (Login/Signup). |
| `tutor-profile` | `TutorProfilePage` | Public profile details for a specific tutor. |
| `payment` | `PaymentPage` | Checkout flow for booking a lesson. |
| `classroom` | `Classroom` | The video lesson interface. |
| `settings` | `SettingsPage` | User account configuration. |
| `referral` | `ReferralPage` | "Refer a Friend" promotional page. |

## Navigation Actions
Exposed via `useNavigation()` hook.

*   `navigate(view: ViewState)`: Switch to a specific page.
*   `goToProfile(tutor: Tutor)`: Open `tutor-profile` and set `activeTutor`.
*   `goToBooking(tutor: Tutor, slot?: string)`: Open `payment` flow.
*   `startSession(session: Session)`: Open `classroom` view.
*   `endSession(redirectView: ViewState)`: Close classroom and return to dashboard.
